# Deep Learning Projects

A compact collection of deep learning examples and experiments, organized into project folders. Contains Jupyter notebooks, training logs, and small apps for image classification, object detection, and related tasks.

Key folders

- Fashion-Image-Classification-with-PyTorch: notebooks and models for fashion image classification.
- Invoice_Classification: experiments for invoice processing and classification.
- Menu_Classification: UI app and models for menu item classification.
- NutriVision-Food-Classification: food classification experiments with run logs.
- Receipt_Object_Detection: object detection examples (YOLO) and training outputs.

Quick start

- Open the relevant `*.ipynb` file in a folder and run the cells.
- If a folder contains `app.py`, run it with:

```bash
python app.py
```

- To install dependencies for a project (if it has `requirements.txt`):

```bash
python -m pip install -r <project_folder>/requirements.txt
```
